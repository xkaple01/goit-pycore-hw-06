## SQLite Virtual Assistant
### Backend and Frontend in Python

SQLite Virtual Assistant uses SQLite database on backend.

OOP Virtual Assistant is completely implemented
in [goit-pycore-hw-07](https://github.com/xkaple01/goit-pycore-hw-07) project.



```sh
git clone https://github.com/xkaple01/goit-pycore-hw-06.git
cd goit-pycore-hw-06
docker build -t xkaple01/goit-pycore-hw-06 .
docker run --rm -it -p 8080:8080 xkaple01/goit-pycore-hw-06
```

### Open browser at 0.0.0.0:8080
